# Book Recommendation Backend

Бэкенд веб-приложения для интеллектуального подбора книг на основе анализа пользовательских предпочтений.

## Используемые технологии

- Python
- FastAPI
- SQLite
- SQLAlchemy
- Swagger / OpenAPI

## Возможности API

- получение списка книг;
- поиск книг по названию, автору и жанру;
- фильтрация книг по жанру;
- получение топа книг;
- получение списка пользователей;
- просмотр избранных книг пользователя;
- просмотр жанровых предпочтений пользователя;
- просмотр истории взаимодействий пользователя с книгами;
- формирование персональных рекомендаций;
- получение общей статистики проекта.
